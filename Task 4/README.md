# Task 4: Predicting Insurance Claim Amounts

## Objective

The objective of this project is to predict medical insurance charges using a Linear Regression model based on customer demographic and health-related information.

## Dataset

- **Dataset:** Medical Cost Personal Dataset
- **Source:** Kaggle

## Tools and Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

1. Loaded and explored the dataset.
2. Checked for missing values and duplicate records.
3. Performed Exploratory Data Analysis (EDA).
4. Encoded categorical variables using Label Encoding.
5. Split the dataset into training and testing sets.
6. Trained a Linear Regression model.
7. Evaluated the model using MAE, RMSE, and R² Score.

## Results

- **Model:** Linear Regression
- **MAE:** 4186.51
- **RMSE:** 5799.59
- **R² Score:** 0.7833

## Conclusion

The Linear Regression model successfully predicted medical insurance charges with good performance. The analysis showed that age, BMI, and smoking status have a significant impact on insurance costs.
