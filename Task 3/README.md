# Task 3: Customer Churn Prediction

## Objective

The objective of this project is to predict whether a bank customer is likely to leave the bank using machine learning techniques.

## Dataset

* **Dataset:** Churn Modelling Dataset
* **Source:** Kaggle

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Loaded and explored the dataset.
2. Removed unnecessary columns.
3. Performed Exploratory Data Analysis (EDA).
4. Encoded categorical variables.
5. Split the dataset into training and testing sets.
6. Trained a Random Forest Classifier.
7. Evaluated the model using Accuracy Score, Confusion Matrix, and Classification Report.
8. Analyzed feature importance.

## Results

* Model: Random Forest Classifier
* Evaluation Metrics:

  * Accuracy Score
  * Confusion Matrix
  * Classification Report
* Feature Importance Analysis

## Conclusion

The Random Forest model successfully predicted customer churn and identified the most influential customer attributes. This project demonstrates a complete machine learning workflow for customer churn prediction.
