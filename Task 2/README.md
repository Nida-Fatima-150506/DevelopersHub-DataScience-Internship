# Task 2: Credit Risk Prediction

## Objective

The objective of this project is to predict whether a loan applicant is likely to receive loan approval based on demographic and financial information using a machine learning classification model.

## Dataset

* **Dataset:** Loan Prediction Dataset
* **Source:** Kaggle

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Loaded and explored the dataset.
2. Handled missing values using mode and median imputation.
3. Removed the `Loan_ID` column.
4. Performed exploratory data analysis (EDA).
5. Encoded categorical variables using Label Encoding.
6. Split the dataset into training and testing sets.
7. Trained a Logistic Regression model.
8. Evaluated the model using Accuracy Score, Confusion Matrix, and Classification Report.

## Results

* Model: Logistic Regression
* Accuracy: **78.86%**

## Conclusion

The Logistic Regression model successfully predicted loan approval status with an accuracy of 78.86%. This project demonstrated the complete workflow of a binary classification problem, including data preprocessing, exploratory data analysis, model training, and performance evaluation.
