# Task 1: Exploring and Visualizing the Iris Dataset

## Objective

The objective of this task is to explore and visualize the Iris dataset using Python. The project focuses on understanding the dataset, performing exploratory data analysis (EDA), and creating visualizations to identify patterns among different iris flower species.

## Dataset

* **Dataset:** Iris Dataset
* **Source:** Kaggle

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Workflow

1. Loaded the dataset.
2. Inspected the dataset using `head()`, `shape`, `columns`, `info()`, and `describe()`.
3. Checked for missing values and duplicate records.
4. Removed the unnecessary `Id` column.
5. Created:

   * Scatter Plot
   * Histogram
   * Box Plot
   * Correlation Heatmap
6. Interpreted the visualizations and summarized key findings.

## Key Insights

* The dataset contains three iris flower species.
* Setosa is clearly distinguishable from the other species.
* Petal length and petal width have a strong positive correlation.
* No missing values were found in the dataset.

## Conclusion

This project provided practical experience in exploratory data analysis (EDA) and data visualization using Python. The visualizations helped identify relationships between flower measurements and distinguish different iris species.
